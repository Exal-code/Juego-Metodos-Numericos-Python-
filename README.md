# Juego de Métodos Numéricos

Proyecto académico desarrollado para la facultad FIME, enfocado en el aprendizaje interactivo de métodos numéricos mediante Python.

## Características Principales
* **Aprendizaje Interactivo:** Implementación de un flujo dinámico de preguntas y respuestas.
* **Lógica de Progresión:** Sistema de validación de respuestas con niveles de dificultad escalables.
* **Generación Dinámica:** Algoritmos para la creación de problemas matemáticos en tiempo real.

## Tecnologías Utilizadas
* **Lenguaje:** Python
* **Conceptos Aplicados:** Estructuras de control, validación de datos y lógica algorítmica.

## Cómo Visualizar el Proyecto
> **Nota:** Este proyecto es una aplicación de consola. Para ejecutarlo:
1. Clona el repositorio.
2. Ejecuta `python main.py` (o el nombre de tu archivo principal).
